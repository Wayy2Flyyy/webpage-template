Developer notes

How to extend and wire this UI showcase.

Structure

This page is intentionally simple: a single HTML file, one CSS file, and one JS file. All sections are marked with IDs so you can jump, scroll, or animate between them.

CSS tokens

Update the :root variables in ui-showcase.css to change the entire visual theme: colours, radii, shadows, and transitions.

JavaScript hooks

The JavaScript file includes ready-to-use hooks for: navigation pills, toggles, sliders, and future animations. Extend them to add real behaviour to your components.
