Developer notes

Integration guidance

This checkout is a front-end shell. Hook it into your actual payment system before using it in production.

  1. Replace the static prices and copy in the Order summary with values from your backend or payment configuration.
  2. Attach the #payButton click handler to your checkout session, payment intent, or server endpoint.
  3. Run full validation on the server side before confirming any charge, including fraud checks and business logic.
  4. Use gateway webhooks (for example, payment_succeeded) to activate subscriptions, deliver products, or unlock access.
  5. If you need help, contact the Mortal Plugins team on Discord: https://discord.gg/dKbvtafN5E .
